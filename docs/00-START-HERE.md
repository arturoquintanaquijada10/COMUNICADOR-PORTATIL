## 🎨 Design System - Quick Reference

### Colores
- **Primary**: #2563EB (Azul)
- **Success**: #10B981 (Verde)
- **Error**: #DC2626 (Rojo)
- **Info**: #06B6D4 (Cyan)
- + 9 colores adicionales para estados y neutrales

### Tipografía
- H1: 48px / 700
- H2: 36px / 600
- H3: 24px / 600
- Body: 16px / 400
- Caption: 12px / 500
- Button: 14px / 600
- Font: Inter

### Componentes
1. **PictoButton** - Botón con imagen y etiqueta
2. **PictoGrid** - Grid responsivo de pictogramas
3. **TextArea** - Área de texto grande
4. **ActionBar** - Barra de acciones
5. **ThemeSelector** - Selector de temas
6. **ConfigModal** - Modal de configuración

### Spacing
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px

### Breakpoints
- Mobile: < 768px (2 columnas)
- Tablet: 768px - 1023px (3 columnas)
- Desktop: ≥ 1024px (4 columnas)
